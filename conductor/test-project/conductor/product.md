# Product Guide

## Initial Concept
A simple task management application for small teams.

## Target Users
- Small development teams (2-10 people)
- Freelancers managing multiple projects
- Startups needing lightweight project tracking

## Goals
- Enable quick task creation and assignment
- Provide clear visibility of project progress
- Support collaboration without complexity

## Key Features
- Task CRUD operations
- User authentication and authorization
- Real-time updates
- Simple kanban board view
